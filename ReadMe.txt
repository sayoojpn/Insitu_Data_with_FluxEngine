Instructions:
1) Firstly make sure this code is run in whichever environment you have installed FluxEngine (as well as the other required packages).

2) The required packages, which can be installed with pip, are:
	- os
	- numpy as np
	- pandas as pd
	- matplotlib as plt
	- seaborn as sns
	- datetime as dt
	- netCDF4
	- basemap-data-hires

3) Ensure all datetime formats are as yyyy-mm-dd hh:mm:ss - especially check if the data has been exported to Excel (or similar software) at any point.
